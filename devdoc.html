Quick Start Guide

The JioCX SMS API is a robust and reliable solution designed to simplify the integration of SMS messaging functionalities into your applications. This developer-friendly API empowers businesses to efficiently communicate with their customers.

Send a simple SMS text by defining destination and message text, or experiment with a rich set of features, including bulk sending. During the Testing period, you'll be able to send 1,000 free SMS messages per day

Prerequisites

1. JioCX account is required to send SMS messages. If you don’t have an account, you can create one at http://developer.jiocx.com/ or by clicking on Sign Up button the Top right coroner of this page.
2. DLT template with messages for customers.
3. A development machine that can run sample code.
